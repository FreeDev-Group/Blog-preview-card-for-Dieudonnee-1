# Blog Preview Card

A simple HTML & CSS project that recreates a blog preview card.

##  Objective
This challenge helps practice:
- HTML structure
- The box model
- Flexbox layout
- Responsive design

##  Technologies
- HTML5
- CSS3

##  Features
- Card with image, title, description, and author
- Clean design with shadows and rounded corners
- Responsive layout for mobile devices

##  What I learned
- How to structure a clean and semantic HTML layout  
- How to use CSS variables for better styling control  
- How to create layouts using Flexbox  
- How to style components (card, buttons, images)  
- How to make a responsive design  

##  Challenges faced
- Creating the offset shadow effect behind the card  
- Maintaining consistent spacing and alignment  
- Making the design responsive across different screen sizes  
- Handling image sizing without distortion  

##  Usage
1. Download the project files  
2. Open `index.html` in your browser  
3. Customize the content as needed
